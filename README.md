# Lak and Point

Beauty/nail service landing page and price-list web prototype.

> **Status:** Frontend prototype / portfolio project.  
> Built as a compact landing page and price-list presentation for a service business scenario.

## What it is

Lak and Point is a visual web prototype for a beauty/nail service. The project focuses on presentation, responsive layout, contact flow and price-list display.

## What it shows

- landing page structure for a small service business
- visual frontend design
- responsive React/Vite implementation
- price-list presentation
- practical portfolio-oriented delivery

## Stack

- React
- JavaScript
- Vite
- GSAP
- CSS modules

## Run locally

```bash
npm install
npm run dev
```

## Notes

This repository is part of my public web portfolio. It is kept as a frontend prototype rather than an actively maintained product.
